# PP-Generator PP-Generator is an [osu!](https://osu.ppy.sh/home) [discord](https://discordapp.com/) bot written in Java on IntelliJ as Maven project. The main idea of this is to show how the bot is working concretely, not necessarily to run it yourself. Nonetheless, feel free to get it going :) If you have any questions, ideas, suggestions, improvements, ..., feel free to either open an issue or just message me (`Badewanne3#0685`) on Discord directly.
